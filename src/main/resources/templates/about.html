About Pathfinder

Pathfinder is a full-stack web application that lets users create, share, and explore travel routes with rich details like GPX coordinates, videos, and image galleries. It’s designed to be responsive and provide a smooth user experience with dynamic updates powered by the Fetch API.

Key Features
  • 🛤️ Add and explore routes by category (Pedestrian, Bicycle, Motorcycle, Car)
  • 📍 Attach GPX coordinates, YouTube videos, and image galleries
  • 🌦️ Live weather forecast integration (OpenWeather API)
  • 👤 User profiles with customization and leveling system
  • 💬 Dynamic interactions without page reloads
  • 👥 Admin panel with user management & statistics
Tech Stack

Backend: Java, Spring Boot, Spring Security
Frontend: Thymeleaf, JavaScript, HTML5, CSS3
Database: MySQL
Integrations: Cloudinary API, OpenWeather API, Chart.js

Roadmap
  • 🐳 Docker support with docker-compose
  • 🔑 Social login (Google, GitHub)
  • 💬 Messaging feature between users
  • 🗑️ Admin ability to delete comments
